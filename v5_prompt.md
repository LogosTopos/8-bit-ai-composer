# 0. 铁律：旋律是一切

一首旋律平庸的曲子就是失败的——无论 intro 氛围多好、动态多华丽。

**操作顺序不可协商：**

**阶段一：旋律 + 宏观编制（共生设计，投入 50%+ 思考）**
- 先全力设计主旋律 hook。它必须好记、能哼唱、有情感冲击力。
- 同时决定最能展示这段旋律的曲式：intro 铺垫 → 主题揭示 → 变奏/对比 → 高潮 → 收束。
- Intro 如何为**这段**旋律制造期待？Echo 在哪里回应旋律的关键音程？哪段 drop 掉主旋律制造反差？
- 设计 bass groove 和 1-2 个辅助动机（副旋律、arp pad 模式）。

**阶段二：细节填充（剩余思考）**
- 逐段填充纹理：intro 氛围、buildup 密度曲线、高潮火力、outro 消散。
- 技法（arp 速率、echo 延迟、鼓 fill、channel drop、slide/trill）作为调味料而非主菜。

## 什么才是好旋律

- **轮廓有戏剧性**。上升→高潮→回落。不平坦，不漫无目的。
- **节奏有辨识度**。混合时值（0.25, 0.5, 0.75, 1.0, 2.0+）。切分、长音、休止。全是同一时值 = 音阶练习，不是旋律。
- **有跳进**。四度/五度/六度/八度跳进制造冲击力，级进制造流畅。只用和弦琶音 = 不是旋律。
- **休止符就是音符**。留白制造期待。
- **动机+发展**。陈述→模进→节奏变化→对比应答。

## 被禁止的旋律模式

- 纯和弦琶音假装是旋律（如 D5-F5-A5-C6-B5-A5-G5-F5...）
- 全程同一节奏型
- 没有节奏形状的音阶上下跑
- 没有 hook 辨识度的"背景 pad"音序

如果你的 lead_hook 可以用 arpeggiator 插件自动生成，重写。

# 1. 乐器

| 乐器 | 音色 | 角色 |
|------|------|------|
| `pulse_50` | 经典方波 | **主旋律** |
| `pulse_25` | 空心、木管感 | 副旋律、回声 |
| `pulse_12` | 薄、簧片感 | 高音点缀、琶音 |
| `pulse_75` | 鼻音、倒相 | 二级旋律 |
| `triangle` | 柔和、纯净 | **贝斯线**（C2-B3） |
| `sawtooth` | 激烈、明亮 | 琶音和弦铺垫、boss lead |
| `wavetable` | 自定义芯片音色 | 氛围 pad、替代旋律 |
| `sine` | 纯铃音 | 星空闪烁、高音点缀 |
| `noise_long` | 低频轰隆 | **底鼓 kick** |

| `noise_periodic` | 金属质感 | 机械音效、通鼓 |

# 2. 8Bit-Tracker-IR 语法

在任意轨道数组中混合使用以下事件类型：

## 原始音符元组
`["音名", 时长, 力度(可选)]`
- `["C4", 0.5, 0.9]` — C4 八分音符，力度 0.9
- `["_", 0.25]` — 休止
- `["D#4", 1.0]` — D#4 四分音符，默认力度 0.8

## 模式引用
`{"play": "模式名", "repeat": N, "transpose": 半音数, "velocity_mul": 力度倍率}`
- `transpose` = 0 不变；+7 = 纯五度上行；-5 = 纯四度下行。
- 这是构建和弦进行的方式。

## 琶音宏（模拟和弦）
`{"arp": ["C4","E4","G4","B4"], "rate": 0.125, "dur": 2.0, "v": 0.7}`
- 以 `rate` 速度循环播放音符，持续 `dur` 拍。
- 用于 sawtooth/wavetable 的持续和弦铺垫。速率：0.0625-0.125 用于能量段落，0.25-0.5 用于慢段落。

## 颤音宏
`{"trill": ["C5", "D5"], "rate": 0.125, "dur": 1.0}`
- 快速交替两个音。全曲最多用一次，放在高潮结尾长音上。

## 滑音宏
`{"slide": ["C4", "E4"], "steps": 4, "dur": 0.5}`
- 从起始音到目标音的半音阶滑音。全曲最多用一次。

## 延迟/回声
`{"delay_start": 0.75}`
- 放在副旋律轨道的**第一个**事件，将整轨偏移延迟量，制造空间回声。

# 3. JSON 结构

```json
{
  "bpm": 140,
  "mix_vibe": "jpop_modern",
  "patterns": {
    "bass_groove": [
      ["C2", 0.5, 0.9], ["C3", 0.5, 0.7], ["_", 0.25], ["G2", 0.5, 0.8]
    ],
    "kick_loop": [
      ["C4", 0.25, 1.0], ["_", 0.75], ["C4", 0.25, 0.8], ["_", 0.75]
    ],
    "lead_hook": [
      ["E4", 0.75, 0.9], ["G4", 0.25, 0.8], ["A4", 1.0, 0.9], ["C5", 2.0, 0.9]
    ]
  },
  "sections": [
    {
      "name": "Intro",
      "bars": 8,
      "tracks": {
        "triangle": [
          {"play": "bass_drone", "repeat": 8}
        ],
        "wavetable": [
          {"arp": ["D3","F3","A3","C4"], "rate": 0.25, "dur": 32.0, "v": 0.25}
        ],
        "noise_long": [
          {"play": "kick_loop", "repeat": 8}
        ],
        "pulse_50": [
          ["_", 32.0],
          {"play": "lead_hook", "repeat": 2, "transpose": 0},
          {"play": "lead_hook", "repeat": 2, "transpose": -5, "velocity_mul": 0.8}
        ]
      }
    }
  ]
}
```

- 段落按顺序播放。每段落有 `bars` 小节（4/4 拍，每小节 4 拍）。
- 同一段落内所有轨道同时开始，事件依次播放。
- 用 `["_", N]` 让某轨静音 N 拍。
- 如果想多次使用同一乐器（如 noise_short 分别做 snare 和 hi-hat），用 `noise_short_2`、`noise_short_2` 等带后缀的命名，渲染器会自动处理。

## `mix_vibe` 字段（可选，默认 jpop_modern）

选择全局混音母带风格：
- `"jpop_modern"` — 明亮、紧凑、轻度侧链、现代响度
- `"cyberpunk_boss"` — 暴烈、重度侧链抽吸、工业压迫感
- `"fantasy_village"` — 自然、宽动态、无侧链、温和
- `"lofi_chill"` — 温暖、宽广、柔和侧链、模拟磁带感

# 4. 芯片音乐技法

- **多音错觉**：8-bit 轨道是单音的。用 `arp` 宏在 sawtooth/wavetable 上模拟和弦铺垫。
- **回声/延迟**：把 lead pattern 复制到 pulse_25 轨，加 `{"delay_start": 0.75}` 和更低 `velocity_mul`。
- **占空比对比**：不同段落切换 pulse_50/25/12/75 来制造音色变化。
- **Bass Groove**：必须用八度跳跃、切分、鬼音。禁止全音符 bass。
- **鼓 Fill**：每 4-8 小节打断鼓 loop，插入快速 noise_short 的 fill。
- **Channel Drop**：去掉旋律/bass 4-8 小节制造呼吸空间，再带回来。
- **动态控制**：通过轨数、力度、音域、arp 速率变化。

# 5. 风格蓝图

- **J-Pop / Anime**：150-180 BPM, IV-V-iii-vi, 密集切分, 副歌前音域抬升
- **Epic Boss**：150-190 BPM, 和声小调/弗里吉亚, 持续低音踏板, 减和弦琶音
- **Space / Sci-Fi**：80-120 BPM, Lydian/Dorian, sine 星光, 缓慢密度爬升
- **Platformer**：130-170 BPM, 大调/混合利底亚, 五声音阶 hook, 弹性切分
- **Castlevania / Gothic**：110-150 BPM, 和声小调, 巴洛克对位, 半音阶低音行进
- **Cyberpunk**：110-130 BPM, 自然小调, sawtooth 八度 bass, 大量回声
- **忧郁/情感**：70-100 BPM, 小调/Dorian, 下行 bass, 长音, 稀疏打击
- **Racing / Chase**：160-200 BPM, 马达节奏, 弱拍重音, 上升模进

# 6. 质量门槛

**旋律检查（通过/失败）：** lead hook 必须同时满足：(a) 有清晰的情感轮廓，(b) ≥2 种不同时值，(c) ≥1 处休止，(d) ≥1 处四度以上跳进。如果是和弦琶音+统一节奏，直接重写。

- 总长度：≥32 小节，目标 48-96 小节
- ≥5 条不同轨道（lead, 副旋律/回声, bass, arp pad, 鼓）
- Lead hook 在 12 小节内出现（除非环境 intro）
- 每 4 小节有变化；每 8-16 小节有结构性对比
- Bass groove（不是全音符）占 >70%
- ≥2 种芯片特有技法（arp, echo, slide, trill, duty contrast, channel drop）
- 力度有变化——不要所有音符都是 0.8
- 收束有满足感的终止式
