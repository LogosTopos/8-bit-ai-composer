# 8-Bit AI Composer

基于 DeepSeek API 的 8-bit 芯片音乐 AI 作曲系统，支持从自然语言描述到成品 MP3 的完整流水线，内置意图驱动自动混音 DSP 引擎。

## 项目结构

```
8-bit-ai-composer/
├── main.py                  # 图形界面入口（src/ui/app.py）
├── demo_v5_cli.py           # V5 命令行快速启动
├── _mix_json.py             # 离线混音工具（从 JSON 文件渲染音频）
├── v5_prompt.md             # 网页端 AI 作曲用 System Prompt
│
├── src/
│   ├── audio/               # 8-bit 合成器 + Automix DSP 引擎
│   │   ├── automix.py       # ★ V5 意图驱动自动混音母带
│   │   ├── synthesizer.py   # 实时合成器
│   │   └── constants.py     # 音色/频率常量
│   │
│   ├── demo/
│   │   ├── llm_client.py    # 共享 LLM 客户端（v1/v2/v4 共用）
│   │   ├── renderer.py      # 离线渲染器（向量化高速版）
│   │   ├── macro_expander.py        # V2 Macro 格式展开器
│   │   ├── tracker_ir_expander.py   # V4 Tracker IR 格式展开器
│   │   │
│   │   ├── demo_v1/         # V1: 基础直出
│   │   ├── demo_v2/         # V2: Macro Pattern 格式
```

## 快速开始

### 1. 安装依赖

```bash
pip install numpy scipy soundfile python-dotenv colorama openai sounddevice
# 或使用 uv:
uv pip install numpy scipy soundfile python-dotenv colorama openai sounddevice
```

MP3 输出需要系统安装 `ffmpeg`。

### 2. 配置 API Key

在项目根目录创建 `.env` 文件：

```
CHAT_API_KEY=sk-your-deepseek-api-key
```

### 3. 启动

```bash
# V5（推荐，旋律优先 + 自动混音）
python demo_v5_cli.py
# 或
python -m src.demo.demo_v5

# V4（Tracker IR，适合氛围型）
python -m src.demo.demo_v4

# V2（简洁 Macro 格式）
python -m src.demo.demo_v2
```

进入后直接输入音乐描述即可。支持的命令：

| 命令 | 作用 |
|------|------|
| 你的音乐描述 | 生成新曲子 |
| 修改意见 | 在上次基础上修改 |
| `reset` | 重置会话 |
| `flash` | 切换到 Flash 模型（更快） |
| `pro` | 切换回 Pro 模型 |
| `vibes` | 查看可用混音风格（V5） |
| `jpop_modern` / `lofi_chill` 等 | 切换混音风格（V5） |
| `exit` | 退出 |

## V5 混音风格

在 V5 中输入 vibe 名称即可切换：

| Vibe | 听感 |
|------|------|
| `jpop_modern` | 明亮、紧凑、现代响度（默认） |
| `cyberpunk_boss` | 暴烈、重度侧链、工业压迫感 |
| `fantasy_village` | 自然、宽动态、温和不刺耳 |
| `lofi_chill` | 温暖、宽广、模拟磁带感 |

## 离线混音（推荐工作流）

用网页端 AI 生成 JSON，再用本地引擎渲染为音频——速度快、省钱、可反复调整混音风格。

### Step 1: 网页端生成 JSON

1. 打开 `v5_prompt.md`，全选复制
2. 粘贴到 DeepSeek / Gemini 网页版对话框
3. 在末尾追加你的需求（例如："写一首太空电梯的 8-bit 音乐，D Dorian，128 BPM，两分钟以上，要有从地面到太空的爬升感"）
4. 发送，AI 会输出一段 JSON
5. 复制 JSON，保存为 `.txt` 文件

### Step 2: 本地渲染

```bash
# 默认 jpop_modern 风格
python _mix_json.py 你的文件.txt

# 指定混音风格
python _mix_json.py 你的文件.txt lofi_chill
```

输出 WAV 和 MP3 会生成在文件同目录。

### Step 3: 换风格重混（不重新生成 JSON）

```bash
python _mix_json.py 你的文件.txt cyberpunk_boss
python _mix_json.py 你的文件.txt fantasy_village
```

同一份 JSON 可以反复渲染不同混音风格，无需重新调用 API。

## 技术架构

### 合成引擎
- 11 种经典 8-bit 波形：pulse(12/25/50/75)、triangle、sawtooth、sine、wavetable、noise(short/long/periodic)
- ADSR 包络、vibrato、arpeggio
- 基于 NumPy 的向量化离线渲染（比逐采样 Python 循环快 5x）

### Automix DSP 引擎（V5）
纯 Python/NumPy/SciPy 实现，零外部插件依赖：

- **6 种混响空间**：dry → room → hall → cathedral → space_ambient → plate_retro（合成 IR 卷积，无需外部 wav 文件）
- **5 种动态塑形**：natural / punchy / compressed / sustained / gated
- **6 种 EQ 预设**：bright / warm / dark / telephony / bass_boost / scooped（RBJ Cookbook 双二阶滤波器）
- **侧链闪避**：kick → bass/pad/harmony
- **总线胶水压缩 + Look-ahead 砖墙限制器**：防止数字爆音
- 乐器→混音角色自动映射（pulse_50→lead/hall, triangle→bass/dry, sawtooth→pad/space_ambient …）

### 8Bit-Tracker-IR 中间格式
LLM 输出的结构化作曲指令，支持：
- 原始音符元组 + 模式引用 + transpose 和弦进行
- 琶音宏（arp）、颤音宏（trill）、滑音宏（slide）
- 多段落顺序播放 + 回声延迟 + Channel Drop
- 确定性展开器转换为渲染器可用的平铺音符格式
