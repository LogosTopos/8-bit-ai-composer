# Examples

This folder contains the four curated example/reference packs kept after cleanup.

## `highspeed_reference_midi_v1/`

Reference MIDI review pack for fast battle/game music density and arrangement roles. Includes MIDI, MP3 review renders, and structural JSON.

## `community_midi_analysis_v1/`

Community MIDI analysis pack. Useful for comparing register, density, and melodic behavior. Treat licensing carefully before broad public distribution.

## `thermocline_v1_reconsidered/`

Earlier Thermocline battle-BGM output. The backing track is useful; the lead is the failure boundary.

Suggested first listens:

- `01_温跃层战斗BGM_v1_master.mp3`
- `03_去掉主旋律_伴奏对比.mp3`
- `stem_mp3/`

## `thermocline_chatb_direct_v1/`

Direct Chat B pass after dropping the melody-only plan. It delayed and softened the lead while raising bass, drums, harmony, and arps.

Suggested first listens:

- `01_温跃层_ChatB_direct_v1_master.mp3`
- `03_bass_drums_only.mp3`
- `04_no_lead_arrangement.mp3`
- `stem_mp3/`
